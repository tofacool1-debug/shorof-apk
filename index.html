Panduan Pasang Aplikasi

1 Instalasi PWA Langsung (Rekomendasi HP)

Buka tautan web aplikasi ini melalui browser Google Chrome (Android) atau Safari (iOS). Klik tombol Pasang kuning yang muncul di bagian kanan atas layar utama Anda.

2 Bungkus Jadi APK (PWABuilder / Cordova)

Jika Anda mengunggah file index.html ini ke tool online seperti PWABuilder.com atau membungkusnya via CLI, format Metadata Manifest pada baris head berkas ini sudah dimodifikasi sehingga terbebas dari kesalahan compile "missing name".
